<h2>💸 Weekly Pricing Model</h2>

<p>

<hr>

<h2>⚙️ How the System Works</h2>
<ul>
    <li>Real-time data is fetched from APIs like IMD and OpenWeatherMap at regular intervals.</li>
    <li>When disruption values exceed thresholds, multiple sources validate before triggering a claim.</li>
    <li>Income loss is calculated using historical average earnings vs actual working hours.</li>
    <li>Fraud detection uses GPS, movement, and device data processed using Isolation Forest.</li>
</ul>
